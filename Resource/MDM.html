哪些类型的客户将最受益于移动设备管理器？

　　以行业应用程序（LOB）的猛烈增长作为代表性地，可以发现现在移动业务市场正在强劲的增长。越来越多的员工需要远程使用自己的移动设备来进行文件，日历和企业应用程序的操作。为了降低其总成本（ TCO ），企业需要通过通过管理公司笔记本电脑等基础设施一样的方式来管理这些移动设备，以保证所有的设备都可以轻松地访问企业网络而不会对企业机密资料造成损坏、泄密。为了解决这一业务需要，微软公司发布了Microsoft System Center Mobile Device Manager 2008(简称MDM)移动设备管理服务器软件产品。MDM 就是一个针对移动设备的全方位，高效的管理解决方案，可与企业现有的Microsoft IT基础架构紧密结合，为企业IT部门提供可靠、低成本和与桌面管理系统一致的移动终端管理，并保证移动设备能够安全地访问企业网络，实现对Windows Mobile® 设备的高效控制。

产品优势

MDM相对于其它移动终端管理来说有如下独特优势：

· 移动VPN。微软在最新的Windows Mobile操作系统中内置了VPN客户端，从而实现无缝的VPN方案。提供其它设备管理软件无法达到的数据传输安全性需求。
· 免移动终端客户端安装。微软将MDM所需要的客户端内置在Windows Mobile中，因此所有的管理都不需要再终端上再安装额外的软件，极大的减少了IT人员的工作量。
· 统一的管理。移动终端的管理整合在微软PC端的管理系统中，IT人员无须另外学习一套软件的使用，可以利用熟悉的管理工具快速掌握管理移动终端的功能。
· 丰富的管理选项，快速部署。完全的产品化，不需要二次开发，安装配置完成就可以满足绝大部分用户的管理需求。

系统需求

· 必须
64位Windows Server ® 2003 SP2, SQL Server 2005,活动目录，CA，组策略管理器，Windows Mobile 6.1设备
· 非必须
邮件服务器,系统管理服务器，系统中心，ISA服务器
· 价格
移动设备管理器的许可规则遵循Microsoft产品许可使用权力中的服务器/客户机访问许可证（CAL）一节。移动设备管理器对于每一台管理服务器需要一个服务器许可证，对于每一个用户或者设备，需要一个CAL。
