首页安装Visual Studio

　　 这部分是针对刚刚接触.Net Micro Framework的初学者的。SDK安装需要有Visual Studio 2008和MSDN。安装完成之后，我们可以在C:\Program Files\Microsoft .NET Micro Framework中找到NETMF的文件，还有一些示例代码会被安装在“我的文档”的\Microsoft .NET Micro Framework\Samples目录下。
软件需求
　　1. 需要Visual Studio 2008 SP1，或者Visual C# 2008 Express Edition。如果没有这两个其中的任何一个，则可以下载90天免费的适用版本。

  • Visual Studio 2008 支持 .NET Micro Framework 3.0. 如果为低版本的开发，则需要安装 Visual Studio 2005.
  • Microsoft 仅支持 C# .如果你是高级用户，那么你可以使用任何 .NET语言, 但是没有调试的能力。
    •
　　2. 接下来就需要.Net Micro Framework。可以从SDK获得（软件开发工具包），或者下载最新免费版本
　　3. 为硬件下载安装SDK。
　　4. 如果有硬件，用USB将硬件和电脑相连。如果安装失败，可以在安装的SDK的子文件夹驱动器中找到所需要的驱动。

创建新项目

使用Visual Studio 创建.NET Micro Framework项目步骤如下。
　　1. 在Visual Studio 的菜单中选择“文件”，然后选择“新建”，出现“新建项目”对话框.

　　2. 在项目类型树控件中，展开“Visual C#”节点。
　　3. 在展开的节点中选择“Micro Framework”。
　　4. 在名称的输入框中，输入项目的名称。
　　5. 在位置的输入框中，指定项目所在的位置，然后单击“确定”按钮。
　　Visual Studio创建项目后，会生成一个名称为MFWindowApplication1.cs的文件，可以在这个文件里添加代码，或者重新命名，或者往项目里添加其他C#文件。

编写新应用

开发.NET Micro Framework托管应用过程和开发其他.NET Framework平台的应用类似，首先创建类、方法和其他程序需要的元素，然后编译并在模拟器上运行，或者安装部署到目标设备上测试和调试。
