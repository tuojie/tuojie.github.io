Platform and environment

　SharePoint 2010 based entirely on the x64 architecture, and no longer contains the x86 version. SharePoint 2010 also determined that the required hardware and software environment:
  • Server hardware must support the x64
  • SharePoint 2010 server operating system must use the Windows Server 2008 x64 or Windows Server 2008 R2 x64
  • SharePoint 2010 server to use the database must be SQL Server 2005 SP2 x64 or SQL Server 2008 x64
　　

To facilitate the SharePoint developers, SharePoint 2010 provides a way to allow developers to install 64-bit Windows Vista and Windows 7 operating systems. In this way, developers can install in their Windows Vista or Windows 7 development environment, using Visual Studio 2010 for SharePoint application development. SharePoint is based .NET / ASP.NET Web application platform technology. SharePoint 2010 on. NET Framework 3.5 SP1 version. Yes, although the development of SharePoint 2010 will mainly use the Visual Studio 2010, but it use and depend on. NET 3.5 SP1, which is not. NET 4.0.
