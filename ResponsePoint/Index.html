Microsoft Response Point
　　Microsoft Response Point 是专为1到50员工小企业开发的电话系统软件，友好的用户管理界面 可以让一个普通用户便捷的安装设置电话系统，而独特的支持语音用户界面支持在客户需要时即时接通。而您需要的，仅仅是一个低成本的并支持传统电话业务和最新VOIP的升级包。Microsoft Response Point 电话系统易于管理和使用，是小企业的最佳选择。

方便安装和管理
　　Microsoft Response Point电话系统包括软件和电话两部分，非常容易安装和管理。此外Response Point系统中还包括一个基于电脑的管理控制台，任何熟悉电脑环境的人都可以用它来修改或管理系统。

　　在使用这个系统的时候，只需按下电话上的“Response Point”按钮然后告诉系统你想给谁打电话就可以打通电话了。比如，某个用户可以说：“帮我接杰夫的工作电话”，然后系统就可以根据Response Point目录中记载的联系人信息中的电话号码拨号。

Microsoft Response Point 三大技术亮点

　　微软的Response Point以一种很实用的电话系统的形式进入了商业电话系统领域，它可以为少于50名雇员的小型企业提供服务。一个典型的Response Point解决方案包括一个Windows XP嵌入技术的基站和几台电话机。基站类似于大型网络路由器，它链接着企业外部电话线和内线网络。基站没有可扩展接口部分：基站的存储都是固态的芯片，它们被放置在一个封闭的空间或者其他位置默默的工作。仅有的软件安装需求就是管理软件程序了。Response Point的产品不会有任何类似统一沟通服务器（UCS）那样的隐含收费。当你够买到这个硬件的时候，它就会为你使用，知道你不再使用它为止，不会有任何附件的费用。

Response Point解决了什么？
　　Response Point用传统的看法来看是使用新老技术的混合体，用户看来是使用Windows XP嵌入技术的手持电话机。Response Point本质是一个盒装的小型电话机系统。OK，就算它是一个大盒子好了。Response Point来自于微软的一些硬件合作伙伴，它的卖点就在这个普及版本套装中：包含一个基站和五个电话机。你可以随时增加电话机并且随着Response Point的推出，在其功能上无缝整合了VoIP技术。 　　

　　Response Point被设计用于少于50名雇员的企业，并且不像基于软件的解决方案那样，如果你认为你不适合这个系统，你也不得不去适应它。也就是说，Response Point的使用体验就像那些大型企业的电话系统一样的亲切，而不像上述那些软件电话系统一样难于使用。你不需要为每个用户进行管理；你仅仅需要的就是当你的办公室位置改变的时候，将电话机放到新的办公桌并且链接上电话线。 Response Point的现实应用

　　一个典型的Response Point解决方案包括一个Windows XP嵌入技术的基站和几个电话机。机战类似与大型网络路由器，它链接着企业外部电话线和内线网络。基站没有可扩展接口部分；基站的存储都是固态的芯片，它们被放置在一个封闭的弓箭或者其他位置默默的工作。仅有的软件安装需求是就是管理软件程序了。它至少需要安装在办公室内的一台电脑中，以便你可以配置电话系统和每一台电话机。安装独立的电话机需要的仅仅是将它们连接到一个以以太网插口中。从简单的手持电话到更多性能的全面功能设计，在Response Point系统中各种各样的应用成为可能。
